## 快速搭建
在[Releases](https://github.com/dengyuhan/magnetW/releases)下载编译好的`war`包，发布到`Tomcat`即可。

>Windows提供了一键启动版，下载[magnetw-windows-2.x.x.zip](https://github.com/dengyuhan/magnetW/releases) 解压即可用
