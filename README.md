## 2.0特性
#### 2.0改动巨大，不支持从1.x升级到2.x，旧版请谨慎使用
>1. 规则文件支持本地文件和网络文件
>2. 支持自定义配置
>3. 支持多种排序
>4. 增加流量统计
>5. 增加搜索预加载
>6. 增加数据API
>7. 增加管理API
>8. 支持Tracker服务器
