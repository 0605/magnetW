## 简介
`magnetW`基于[magnetX](https://github.com/youusername/magnetX)的规则原理，将各个磁力站的搜索结果统一格式化

## 安装
从[Github Releases](https://github.com/dengyuhan/magnetW/releases)下载对应平台

或者

从[文档页](http://magnetw.app/)下载对应平台
